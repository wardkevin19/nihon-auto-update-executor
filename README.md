## What Is Nihon Executor?

Nihon Executor is a Windows desktop utility for running custom Lua scripts inside Roblox sessions. It emphasizes speed and ease of use, with one-click injection plus a curated script hub that bundles hundreds of scripts across many game types. No activation key or login is needed, so it is simple to open and start using.

The app also includes a persistent queue for running several scripts one after another, and an auto-update system that helps it stay aligned with the newest Roblox client changes. With support for multiple interface languages and a very small footprint, Nihon Executor is aimed at users who want a lightweight and accessible script execution workflow.
